	Состоит из пикселей
	Имеет 3 параметра: ширина, высота, число цветовых каналов
		3 = RGB

## Обработка изображения
1. Размытие
2. Усиление резкости
3. Усиление контраста
4. Удаление шума
5. Нормализация
**Нормализация**
	$F_{\text{norm}} = \frac{F - \min(F)}{\max(f) - \min(f)}$
	$F_{\text{norm}} \in \left[0, 1\right]$
