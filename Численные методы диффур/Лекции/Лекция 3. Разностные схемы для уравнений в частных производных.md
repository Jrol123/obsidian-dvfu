# Гиперболические
**Гиперболический тип**
	$b^2-ac>0$
**Канонический вид**
	$u_{\xi\eta}=F(\xi,\eta, u, u_\xi, u_\eta)$

**Параболический тип**
	$b^2-ac=0$
**Канонический вид**
	$u_{\eta\eta}=F(\xi,\eta, u, u_\xi, u_\eta)$

**Эллиптический тип**
	$b^2-ac<0$
**Канонический вид**
	$u_{\xi\xi}+u_{\eta\eta}=F(\xi,\eta, u, u_\xi, u_\eta)$
