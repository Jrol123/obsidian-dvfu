# Реферат: Тензор инерции (вывод). Эффект Джанибекова

## Введение
В теоретической механике динамика твердого тела существенно зависит от распределения массы относительно осей вращения. Мерой этого распределения служит тензор инерции. Особый интерес представляет случай свободного вращения (отсутствие внешних моментов сил), где на первый план выходит геометрия масс. Эффект Джанибекова (или теорема о теннисной ракетке) является наглядной демонстрацией неустойчивости вращения вокруг средней главной оси инерции, что напрямую следует из структуры тензора инерции и уравнений Эйлера.

---

## 1. Тензор инерции: вывод и свойства

### 1.1 Кинетический момент твердого тела
Рассмотрим твердое тело как систему материальных точек с массами \(m_i\) и радиус-векторами \(\mathbf{r}_i\) в неподвижной системе координат. Введем систему отсчета, жестко связанную с телом, с началом в точке \(O\) (полюсе). Абсолютная скорость точки:
\[
\mathbf{v}_i = \mathbf{v}_O + \boldsymbol{\omega} \times \mathbf{r}_i',
\]
где \(\boldsymbol{\omega}\) — мгновенная угловая скорость, \(\mathbf{r}_i'\) — радиус-вектор точки относительно полюса \(O\) в связанной системе.

Кинетический момент относительно полюса \(O\):
\[
\mathbf{L}_O = \sum_i \mathbf{r}_i' \times m_i \mathbf{v}_i.
\]
Подставляя \(\mathbf{v}_i\) и полагая \(\mathbf{v}_O = 0\) (полюс покоится или совпадает с центром масс), получаем:
\[
\mathbf{L}_O = \sum_i m_i \mathbf{r}_i' \times (\boldsymbol{\omega} \times \mathbf{r}_i') = \sum_i m_i \left[ \omega (\mathbf{r}_i' \cdot \mathbf{r}_i') - \mathbf{r}_i' (\mathbf{r}_i' \cdot \boldsymbol{\omega}) \right].
\]

### 1.2 Определение тензора инерции
В компонентной записи (индексы \(k, l, m = 1,2,3\)):
\[
L_k = \sum_i m_i \left( \delta_{km} r_i^2 - x_{ik} x_{im} \right) \omega_m = I_{km} \omega_m,
\]
где \(r_i^2 = x_{i1}^2 + x_{i2}^2 + x_{i3}^2\).

Тензор инерции второго ранга:
\[
I_{km} = \sum_i m_i \left( r_i^2 \delta_{km} - x_{ik} x_{im} \right).
\]
Для непрерывного распределения массы с плотностью \(\rho(\mathbf{r})\):
\[
I_{km} = \int_V \rho(\mathbf{r}) \left( r^2 \delta_{km} - x_k x_m \right) dV.
\]

### 1.3 Компоненты тензора
В декартовых координатах (\(x, y, z\)) диагональные компоненты — моменты инерции:
\[
I_{xx} = \int (y^2 + z^2) dm, \quad I_{yy} = \int (x^2 + z^2) dm, \quad I_{zz} = \int (x^2 + y^2) dm.
\]
Недиагональные компоненты — центробежные моменты инерции:
\[
$$I_{xy} = -\int xy \, dm, \quad I_{xz} = -\int xz \, dm, \quad I_{yz} = -\int yz \, dm.$$
\]

### 1.4 Свойства тензора
1. **Симметричность**: $I_{km} = I_{mk}$.
2. **Положительная определенность**: для любого ненулевого $\boldsymbol{\omega}$ кинетическая энергия вращения $$T = \frac{1}{2} I_{km} \omega_k \omega_m > 0$$
3. **Преобразование при повороте координат**: компоненты преобразуются как тензор второго ранга:
$$I'_{pq} = a_{pi} a_{qj} I_{ij}$$
где $a_{pi}$ — матрица поворота.
4. **Существование главных осей**: вследствие симметрии существует ортонормированный базис, в котором тензор диагоналей:
$$\mathbf{I} = \begin{pmatrix}
I_1 & 0 & 0 \\
0 & I_2 & 0 \\
0 & 0 & I_3
\end{pmatrix}$$
где $I_1, I_2, I_3$ — главные моменты инерции. Оси называются главными осями инерции. Для тела с произвольной формой главные оси можно найти, решив задачу на собственные значения $I \mathbf{e} = \lambda \mathbf{e}$.

## 2. Эффект Джанибекова (теорема о теннисной ракетке)

### 2.1 Постановка задачи
Эффект Джанибекова — это явление неустойчивости свободного вращения твердого тела вокруг оси, соответствующей среднему по величине главному моменту инерции. Впервые оно было замечено космонавтом В. Джанибековым в невесомости, но математически описано как часть решения уравнений Эйлера для твердого тела.

Рассмотрим твердое тело, закрепленное в центре масс (или свободное в невесомости), на которое не действуют внешние моменты сил. Уравнения движения в главных осях (уравнения Эйлера) имеют вид:
$$\begin{cases}
I_1 \dot{\omega}_1 = (I_2 - I_3) \omega_2 \omega_3, \\
I_2 \dot{\omega}_2 = (I_3 - I_1) \omega_3 \omega_1, \\
I_3 \dot{\omega}_3 = (I_1 - I_2) \omega_1 \omega_2.
\end{cases}$$

### 2.2 Анализ устойчивости стационарных вращений
Стационарное вращение соответствует $\dot{\boldsymbol{\omega}} = 0$. Пусть, например, $\omega_1 = \Omega, \omega_2 = \omega_3 = 0$ (вращение вокруг главной оси 1). Исследуем устойчивость, задав малые возмущения: $\omega_2, \omega_3 \ll \Omega$. Линеаризуем уравнения:
$$I_2 \dot{\omega}_2 = (I_3 - I_1) \Omega \omega_3, \quad I_3 \dot{\omega}_3 = (I_1 - I_2) \Omega \omega_2$$
Продифференцируем первое и подставим второе:
$$\ddot{\omega}_2 = \frac{(I_3 - I_1)(I_1 - I_2)}{I_2 I_3} \Omega^2 \omega_2$$
Обозначим $\lambda = \dfrac{(I_3 - I_1)(I_1 - I_2)}{I_2 I_3} \Omega^2$.

Устойчивость требует $\lambda < 0$ (колебательный режим).
Рассмотрим три случая упорядочения моментов инерции: $I_1 < I_2 < I_3$.

1. **Вращение вокруг оси 1 (наименьший момент)**.  
   $I_1 - I_2 < 0, I_3 - I_1 > 0 \to$ произведение $(I_3 - I_1)(I_1 - I_2) < 0 \implies \lambda < 0\implies$ **устойчиво**.

2. **Вращение вокруг оси 3 (наибольший момент)**.  
   $I_3 - I_1 > 0, I_1 - I_2 < 0 \to$ произведение $(I_3 - I_1)(I_1 - I_2) < 0 \implies \lambda < 0 \to$  **устойчиво**.

3. **Вращение вокруг оси 2 (средний момент)**.  
   $I_2 - I_1 > 0, I_3 - I_2 > 0 \to$ произведение $(I_3 - I_2)(I_2 - I_1) > 0 \implies \lambda > 0 \to$ **неустойчиво**.

Таким образом, вращение вокруг оси со средним главным моментом инерции является неустойчивым: малые возмущения экспоненциально нарастают, приводя к периодическому «кувырканию» тела.

### 2.3 Геометрическая интерпретация
Эффект можно наглядно представить с помощью эллипсоида инерции. В отсутствие моментов сохраняются:
- Кинетическая энергия $T = \dfrac{1}{2} (I_1 \omega_1^2 + I_2 \omega_2^2 + I_3 \omega_3^2)$;
- Квадрат момента импульса $L^2 = I_1^2 \omega_1^2 + I_2^2 \omega_2^2 + I_3^2 \omega_3^2$.

Траектория конца вектора \(\boldsymbol{\omega}\) лежит на пересечении эллипсоида энергии (эллипсоид Пуансо) и сферы момента. При \(I_1 < I_2 < I_3\) точки пересечения, соответствующие вращению вокруг оси 2, лежат на сепаратрисах, разделяющих области устойчивых движений вокруг осей 1 и 3. Любое сколь угодно малое отклонение приводит к переходу от одной устойчивой оси к другой.

### 2.4 Демонстрация и практическое значение
В условиях невесомости (МКС) эффект проявляется как периодические перевороты вращающегося предмета (гайка, ракетка) на 180°. Период переворота зависит от разности моментов инерции и начальной угловой скорости. Этот эффект необходимо учитывать при проектировании космических аппаратов и ориентации спутников, чтобы избежать неконтролируемого вращения.

## Заключение
В работе рассмотрен вывод тензора инерции как фундаментальной характеристики распределения массы твердого тела. Показано, что он является симметричным тензором второго ранга, диагонализация которого приводит к понятию главных осей инерции. На основе уравнений Эйлера выполнен анализ устойчивости вращения свободного тела, который выявил существование неустойчивого стационарного вращения вокруг оси со средним моментом инерции. Это явление, известное как эффект Джанибекова, демонстрирует нетривиальную динамику даже в отсутствие внешних сил и имеет важное прикладное значение в космонавтике.
