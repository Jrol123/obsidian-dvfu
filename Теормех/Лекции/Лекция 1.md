# Принцип наименьшего действия (Принцип Гамильтона)
Связи понижают размерность пространства. 

**Число степеней свободы**
Число переменных, которыми описывается система

**Обобщенные координаты**
Любые независимые между собой параметры $\{q_i\}_{i=1}^s$, однозначно определяющие положение механической системы в пространстве.
- $s$ = число степеней свободы
- Для однозначного определения состояния системы вводят **обобщенные скорости** ($\dot q_i$)

**Энергетический (вариационный) подход**
$q_{0}\left( t \right)$ - траектория движения
Рассматриваем вариации траектории, выбирая ту, на которой функционал $S = \int \limits_{ t_{1} }^{ t_{2} } L\left( q, \dot{q}, t \right)dt$ минимален
