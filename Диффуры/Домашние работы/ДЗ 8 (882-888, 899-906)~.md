# 899
$$\begin{cases}
{\dot x = 2xy-x+y} \\
{\dot y = 5x^4 + y^3 + 2x - 3y}
\end{cases}$$
$$\begin{cases}
{\dot x = -x+y} \\
{\dot y = 2x-3y}
\end{cases}$$
$$\begin{pmatrix}
{-1-\lambda} && {1} \\
{2} && {-3-\lambda}
\end{pmatrix}$$
$$\lambda^2 + 4\lambda + 1 = 0$$
$$\lambda = -2\pm\sqrt{3}$$
Нулевое решение асимптотически устойчиво
