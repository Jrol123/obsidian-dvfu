1. При $x \neq s$, если удовлетворяет однородному уравнению

2. При $x = x_{0}, x = x_{1}$ удовлетворяет краевым условиям
3. При $x = s$

   $G'(s + 0, s) = G'\left( s - 0, s \right) + \frac{1}{a_{0}(x)}$
