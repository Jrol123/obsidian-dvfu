# 1
## Условие
Доказать, что последовательность $\left\{ x_{n} \right\}$ имеет предел.
Найти его, если $x_{n} = \underbrace{ \sqrt[3]{ 6 + \sqrt[3]{ 6 + \ldots + \sqrt[3]{ 6 } } } }_{ n \ \text{раз} }$ для $n \geq 1$
## Решение
### Существование предела
По Теореме Вейерштрасса, если числовая последовательность монотонно возрастает и ограничена сверху, то она имеет конечный предел.

**Ограниченность сверху**
$x_{n} < 2$

*База индукции:*
$x_{1} = \sqrt[3]{ 6 }$
$6 < 8 \implies \sqrt[3]{ 6 } < \sqrt[3]{ 8 } = 2 \implies x_{1} < 2$

*Индукционный переход:*
$x_{k} < 2$
$x_{k + 1} = \sqrt[3]{ 6 + x_{k} } < \sqrt[3]{ 6 + 2 } = \sqrt[3]{ 8 } = 2$

Следовательно, $0 < x_{n} < 2$ для любого $n$

**Монотонность**
$x_{n + 1} \overset{?}> x_{n}$ для любого $n$

*База индукции*
$x_{2} = \sqrt[3]{ 6 + \sqrt[3]{ 6 } }$
Поскольку $\sqrt[3]{ 6 } > 0$, имеем $6 + \sqrt[3]{ 6 } > 6 \implies \sqrt[3]{ 6 + \sqrt[3]{ 6 } } > \sqrt[3]{ 6 }$. $x_{2} > x_{1}$

*Индукционный переход*
$x_{k} > x_{k - 1}$
$x_{k + 1} = \sqrt[3]{ 6 + x_{k} } > \sqrt[3]{ 6 + x_{k-1} } = x_{k}$
Функция $f(t) = \sqrt[3]{6 + t}$ возрастает монотонно, потому что показатель степени $\dfrac{1}{3}> 0$
Следовательно, последовательность монотонно возрастает.

### Нахождение предела
$\lim \limits_{ n \to \infty }x_{n} = A$
Перейдём к пределу $x_{n + 1} = \sqrt[3]{ 6 + x_{n} }$:
$A = \sqrt[3]{ 6 + A }$
$A^{3} = 6 + A \implies A^{3} - A - 6 = 0$
Заметим, что $A = 2$ — корень.
$A^{3} - A - 6 = \left( A - 2 \right)\left( A^{2} + 2A + 3 \right) = 0$
$D_{A^{2} + 2A + 3} = -8 < 0 \implies$ нет действительных корней.

Итого,
Предел $\lim \limits_{ n \to \infty }x_{n} = A = 2$.

## Условие
Вычислить предел
$\lim \limits_{ x \to 1 }\left( \dfrac{x^{n} + x^{n - 1} + \ldots + x^{2} + x - n}{x - 1} \right)$
## Решение
При подстановке $x = 1$ числитель обращается в $\left( \sum \limits_{ k=1 }^{ n }1  \right) - n = 0$, знаменатель в $1 - 1 = 0$
Получается $\dfrac{0}{0}$.

Запишем числитель как сумму разностей (разобьём $n$ на единицы)
$x^{n} + x^{n - 1} + \ldots + x - n = \left( x^{n} - 1 \right) + \left( x^{n - 1} - 1 \right) + \ldots + \left( x - 1 \right)$
$\dfrac{x^{n} + x^{n - 1} + \ldots + x^{2} + x - n}{x - 1} = \dfrac{x^{n}-1}{x-1} + \dfrac{x^{n - 1}-1}{x-1} + \ldots + \dfrac{x-1}{x-1}$
Для каждого $k = 1, 2, \ldots, n$ используем предел $\lim \limits_{ x \to 1 }\dfrac{x^{k}-1} {x-1} \overset{ \text{Лопиталь} }{ = } \lim \limits_{ x \to 1 } \dfrac{kx^{k - 1}}{1} =k$
Получаем:
$\lim \limits_{ x \to 1 }\sum \limits_{ k = 1 }^{ n } \dfrac{x^{k}-1}{x - 1} = \sum \limits_{ k = 1 }^{ n }k = 1 + 2 + \ldots + n = \dfrac{n\left( n + 1 \right)}{2}$

Итого:
$\lim \limits_{ x \to 1 }\left( \dfrac{x^{n} + x^{n - 1} + \ldots + x^{2} + x - n}{x - 1} \right) = \dfrac{n\left( n + 1 \right)}{2}$

## Условие
Найти касательную к графику функции в точке $x = 0$
$f\left( x \right) = \begin{cases}x^{2} + 1 + e^{-\frac{1}{x^{2}}} & x \neq 0 \\ 1 & x = 0\end{cases}$
## Решение
Уравнение касательной к графику функции в точке $x = 0$ имеет вид:
$y = f\left( 0 \right) + f'\left( 0 \right)\left( x - 0 \right)$
Найдём $f\left( 0 \right) = 1$ и вычислим производную $f'\left( 0 \right)$
$f'\left( 0 \right) = \lim \limits_{ h \to 0 } \dfrac{f\left( 0 + h \right) - f\left( 0 \right)}{h} = \lim \limits_{ h \to 0 } \dfrac{h^{2} + 1 + e^{-\frac{1}{h^{2}}} - 1}{h} = \lim \limits_{ h \to 0 }\dfrac{h^{2} + e^{-\frac{1}{h^{2}}}}{h} = \lim \limits_{ h \to 0 }\left( h + \dfrac{e^{-\frac{1}{h^{2}}}}{h} \right)$
Первое слагаемое $h \to 0$ в пределах слева и справа.

Рассмотрим пределы второго слагаемого:

**Предел слева**
Пусть $h < 0$. 
$t = -h \quad  \left( t > 0 \right)$
$t \to +0$ при $h \to -0$
Тогда $h^{2} = t^{2}, e^{-\frac{1}{h^{2}}} = e^{-\frac{1}{t^{2}}}$. Получаем: $\dfrac{e^{-\frac{1}{h^{2}}}}{-h} = \dfrac{e^{-\frac{1}{t^{2}}}}{t} = -\dfrac{1}{te^{\frac{1}{t^{2}}}}$
$\lim \limits_{ t \to +0 }te^{\frac 1 {t^{2}}}= \lim \limits_{t\to{+0}}{\dfrac{e^{\frac{1}{t^{2}}}}{\frac{1}{t}}} = \lim \limits_{t\to{+0}}{\dfrac{2e^{\frac{1}{t^{2}}}}{t}} = +\infty$.
Таким образом, $\lim \limits_{ h \to -0 } \dfrac{e^{-\frac{1}{h^{2}}}}{h} = 0$. Следовательно $\lim \limits_{ h \to -0 } \dfrac{f( 0 + h ) - f\left( 0 \right)}{h} = 0$

**Предел справа**
Пусть $h > 0$
$t = h \quad \left( t > 0 \right)$
$t \to +0$ при $h \to +0$

$\dfrac{e^{-\frac1{h^2}}}{h} = \dfrac{e^{-\frac{1}{t^{2}}}}{t} = \dfrac{1}{t e^{\frac{1}{t^{2}}}} \to 0$
Следовательно, $\lim \limits_{x \to 0^+} \dfrac{f(x) - 1}{x} = 0 + 0 = 0$

Односторонние пределы совпадают и равны нулю, поэтому производная существует: $f'(0) = 0$
Таким образом, касательная горизонтальна и $y = 1 + 0\left( x - 0 \right) = 1$

## Условие
Найти производную $n$-го порядка функции
$y = \dfrac{3 - 2x^{2}}{2x^{2} + 3x - 2}$
## Решение
Выделим целую часть, так как степени числителя и знаменателя равны:
$y = \dfrac{3 - 2x^2}{2x^2 + 3x - 2} = -1 + \dfrac{3x + 1}{2x^2 + 3x - 2}$
Разложим знаменатель на множители: $2x^2 + 3x - 2 = (2x - 1)(x + 2)$.  
Представим дробь в виде суммы простейших:
$\dfrac{3x + 1}{(2x - 1)(x + 2)} = \dfrac{A}{2x - 1} + \dfrac{B}{x + 2}$
Приводя к общему знаменателю и приравнивая числители, находим $A = 1$, $B = 1$.
Следовательно,
$y = -1 + \dfrac{1}{2x - 1} + \dfrac{1}{x + 2}$

Формула $n$-й производной для $(ax + b)^{-1}$:

Рассмотрим функцию $f(x) = (ax+b)^{-1}$. Покажем закономерность.

**Первая производная:**
$f'(x) = -1 \cdot (ax+b)^{-2} \cdot a = (-1) \cdot 1! \cdot a^1 \cdot (ax+b)^{-2}$
**Вторая производная:**
Дифференцируем $f'(x)$:
$f''(x) = (-1) \cdot a \cdot \left( -2 (ax+b)^{-3} \cdot a \right) = (-1)(-2) \cdot a^2 \cdot (ax+b)^{-3} = (-1)^2 \cdot 2! \cdot a^2 \cdot (ax+b)^{-3}$

**База индукции $(n=1)$:**
$f'(x) = \frac{d}{dx}\left( (ax+b)^{-1} \right) = -1 \cdot (ax+b)^{-2} \cdot a = (-1)^1 \cdot 1! \cdot a^1 \cdot (ax+b)^{-2}$
Формула верна для $n=1$.

**Индукционное предположение:**
Пусть формула верна для некоторого $n = k$, то есть
$f^{(k)}(x) = (-1)^k \cdot k! \cdot a^k \cdot (ax+b)^{-(k+1)}$

**Индукционный переход $(n = k+1)$:**
Найдём $k+1$-ю производную, продифференцировав $f^{(k)}(x)$:
$f^{(k+1)}(x) = \dfrac{d}{dx} \left[ (-1)^k \cdot k! \cdot a^k \cdot (ax+b)^{-(k+1)} \right]$
Постоянный множитель $(-1)^k k! a^k$ выносим за знак производной:
$f^{(k+1)}(x) = (-1)^k \cdot k! \cdot a^k \cdot \dfrac{d}{dx} \left( (ax+b)^{-(k+1)} \right)$
Производная степенной функции:
$\dfrac{d}{dx} (ax+b)^{-(k+1)} = -(k+1) \cdot (ax+b)^{-(k+2)} \cdot a$
Подставляем:
$f^{(k+1)}(x) = (-1)^k \cdot k! \cdot a^k \cdot \left( -(k+1) a (ax+b)^{-(k+2)} \right)$
Упрощаем знаки и степени:
$f^{(k+1)}(x) = (-1)^k \cdot (-1) \cdot k! \cdot (k+1) \cdot a^{k+1} \cdot (ax+b)^{-(k+2)}$
Замечаем, что $(-1)^k \cdot (-1) = (-1)^{k+1}$ и $k! \cdot (k+1) = (k+1)!$.
Получаем:
$f^{(k+1)}(x) = (-1)^{k+1} \cdot (k+1)! \cdot a^{k+1} \cdot (ax+b)^{-(k+2)}$
Это в точности соответствует формуле для $n = k+1$.

Итого:
$\dfrac{d^n}{dx^n} \left( (ax + b)^{-1} \right) = (-1)^n \cdot n! \cdot a^n \cdot (ax + b)^{-(n+1)}$

Применяем её к каждому слагаемому:
- Для $\dfrac{1}{2x - 1}  \quad\left( a = 2 \right)$:
  $\dfrac{d^n}{dx^n} \left( \dfrac{1}{2x - 1} \right) = \left( -1 \right)^n \cdot n! \cdot 2^n \cdot \left( 2x - 1 \right)^{-\left( n+1 \right)}$
- Для $\dfrac{1}{x + 2} \quad \left( a = 1 \right)$:
  $\dfrac{d^n}{dx^n} \left( \dfrac{1}{x + 2} \right) = (-1)^n \cdot n! \cdot 1^n \cdot (x + 2)^{-(n+1)} = (-1)^n \cdot n! \cdot (x + 2)^{-(n+1)}$
Производная константы $-1$ равна $0$ для $n \geq 1$.

Складывая результаты, получаем:
$y^{(n)} = (-1)^n \cdot n! \cdot \left( \dfrac{2^n}{(2x - 1)^{n+1}} + \dfrac{1}{(x + 2)^{n+1}} \right)$
