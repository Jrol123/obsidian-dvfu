# ТЗ
Исследовать влияние сопротивления воздуха на траекторию
движения тела, брошенного под углом к горизонту, и освоить методы
численного решения дифференциальных уравнений движения.
# Задачи
1. Реализовать численное интегрирование уравнений движения методом Эйлера
2. Создать сравнительный анализ траекторий с сопротивлением и без
сопротивления воздуха
3. Исследовать влияние параметров на траекторию:
	1. Коэффициент сопротивления воздуха
	2. Начальная скорость и угол броска
	3. Масса тела
4. Разработать интерактивный интерфейс для управления параметрами и
визуализации результатов
