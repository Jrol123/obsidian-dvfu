# Задание 1
Исходная физическая модель: Классическая модель Лоренца (1963 г.) является упрощённой моделью конвекции в слое жидкости, нагреваемой снизу. Она описывает эволюцию трёх безразмерных переменных:
- $x\left( t \right)$ — скорость вращения конвективных валов.
- $y\left( t \right)$ — разность температур между восходящими и нисходящими потоками.
- $z\left( t \right)$ — отклонение вертикального профиля температуры от линейного.
Система уравнений:
{█(x^'=-σx+σy@y^'=rx-y-xz@z^'=xy-bz)┤
где σ — число Прандтля, ρ — нормированное число Рэлея, b — геометрический параметр.
Модификация: В реальном климате на систему действуют внешние периодические силы (например, сезонные изменения солнечного нагрева). Смоделируйте это, добавив периодическое возмущение к параметру r, который отвечает за разность температур между нижним и верхним слоями.
Пусть ρ(t)=ρ0+Acos(ωt), где:
  ρ0 — среднее значение ("средняя солнечная интенсивность").
  A — амплитуда сезонных колебаний (0≤A<ρ0).
  ω — частота сезонных колебаний.
Задания для исследования:
  Анализ стационарных точек: Для системы без сезонов (A=0) найдите стационарные точки и условие их устойчивости.
  Численное моделирование:
  Возьмите стандартные параметры, ведущие к хаотическому аттрактору: σ=10, ρ0=28, b=8/3.
  Зафиксируйте частоту ω=2π/T, где период T выберите сопоставимым со временем "оборота" траектории вокруг одной из стационарных точек (например, T≈1 безразмерных единиц времени).
  Проведите численное интегрирование для разных значений амплитуды A: A=0,2,5,10.
  Постройте фазовые портреты (x,z) и временные ряды y(t) для каждого случая.
  Качественный анализ результатов:
  Опишите, как слабые сезонные колебания (A=2) влияют на классический аттрактор Лоренца. Меняется ли картина в плоскости (x,z)? Что происходит с временным рядом?
