## Хищник-Жертва (Модель Лотки — Вольтерры)
$$
\cases{
\dot x = (\alpha - \beta y) x \\
\dot y = (-\gamma + \delta x) y
}
$$
$x$ - жертва
$y$ - хищник

$$
\cases{
(\alpha - \beta y) x = 0 \\
(-\gamma + \delta x) y = 0
}
$$
**Тривиальная**
$$
x = y = 0
$$
**Нетривиальная**
$$
\cases{
\alpha - \beta y = 0 \\
-\gamma + \delta x = 0
} \implies \cases{
y= \dfrac{\alpha}{\beta} \\
x = \dfrac{\gamma}{\delta}
}
$$
