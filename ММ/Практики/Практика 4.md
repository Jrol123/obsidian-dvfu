$$
\cases{
\dot N_1 = a_1 N_1 + b_{12} N_1 N_2 - c_1 N_1 ^ 2 \\
\dot N_2 = a_2 N_2 + b_{21} N_1 N_2 - c_2 N_2 ^ 2
}
$$

**Варианты**
Симбиоз - $b_{12} > 0, b_{21} > 0$
Хищник жертва - $b_{12} \cdot b_{21} < 0$
Комменсализм - $b_{12} > 0, b_{21} = 0$
$b_{12} = 0, b_{21} < 0$
Нейтрализм - $b_{12}=b_{21}=0$
Взаимная конкуренция $b_{12} < 0, b_{21} < 0$

## Модель 
$$
\cases{
\dot N_1 = N_1\left( r_1 - \beta_1 N_1 - \alpha_2 N_2 \right)  \\
\dot N_2 = N_2\left( r_2 - \beta_2 N_2 - \alpha_1 N_1 \right)
}
$$
$r_{1,2}; \beta_{1, 2}; \alpha_{1,2} > 0$

**Стационарные точки**
1. $N_1 = N_2 = 0$
2. $N_1 = 0, N_2= \dfrac{r_2}{\beta_2}$
3. $N_1 = \dfrac{r_1}{\beta_1}, N_2 = 0$
4. $N_1 = \dfrac{\alpha_1 \beta_2 - \alpha_2 r_2}{\beta_1 \beta_2 - \alpha_1 \alpha_2}, N_2 = \dfrac{\beta_1 r_2 - \alpha_1 r_1}{\beta_1 \beta_2 - \alpha_1 \alpha_2}$


$$
\cases{
N_2 = \dfrac{r_1}{\alpha_2} - \dfrac{\beta_1}{\alpha_2} N_1 \\
N_2 = \dfrac{r_2}{\beta_2} - \dfrac{\alpha_1}{\beta_2} N_1
}
$$
