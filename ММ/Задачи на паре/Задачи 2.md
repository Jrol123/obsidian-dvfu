# Модель Лотки-Вольтерры
## Задание
Популяции рысей ($y\left( t \right)$) и зайцев ($x\left( t \right)$) в заповеднике требуют управления. Классическая модель показала себя слишком идеалистичной. Вам, как математическому экологу, поручили усовершенствовать модель, учтя следующие реальные факторы:
1.  Ограниченность ресурсов для жертв: Емкость среды для зайцев не бесконечна (например, ограниченность кормовой базы).
2.  Внутривидовая конкуренция хищников: Рыси могут конкурировать друг с другом за лучшие территории, не связанные напрямую с едой.

## Решение
### 1
Модель Лотки-Вольтерры в чистом виде:
$$
\cases{
\dot x = (\alpha - \beta y) x \\
\dot y = (-\gamma + \delta x) y
}
$$
Нужно добавить ограничение по объёму $P$ как в логистической модели: $\dfrac{dP}{dt} = rP\left( 1 - \dfrac{P}{k} \right)$.
Для этого добавим ёмкость среды $k$.

### 2
Для внутривидовой конкуренции хищников добавим член $-\sigma y^{2}$, т. е. большое количество рысей мешает появлению ещё большему количеству рысей.

Результат:
$$
\begin{cases}
\dot{x} = \left( \alpha \left( 1 - \dfrac{x}{k} \right) - \beta y \right) x \\
\dot{y} = \left( -\gamma + \delta x - \sigma y \right) y
\end{cases}
$$
