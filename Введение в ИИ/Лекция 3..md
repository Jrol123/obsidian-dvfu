$E_{D} \left[ \left( y - \hat f (x) \right)^{2} \right]$
$y = f\left( x \right) + \epsilon  \quad  \quad E\left[ \epsilon \right] = 0 \quad Var\left( \epsilon \right) = \sigma^{2}$

$E_{D}\left[ \left( f\left( x \right) + \epsilon - \hat{f}\left( x \right) \right)^{2} \right] = E_{D} \left[ \left( f\left( x \right) - \hat{f}\left( x \right) \right)^{2} + 2 \left( f\left( x \right) - \hat{f}\left( x \right) \right)\epsilon + \epsilon^{2} \right]$
	$\pm E_{D}\left[ \hat{f}\left( x \right) \right]$ (Можем вытащить $f\left(x\right)$ т. к. это константа)
$\left( f\left( x \right) - E_{D} \left[ \hat{f}\left( x \right) \right] \right)^{2} + E_{D}\left[ (\hat f(x) - E_D\left[ \hat f(x) \right]^2)^2 \right] + \sigma^2$
