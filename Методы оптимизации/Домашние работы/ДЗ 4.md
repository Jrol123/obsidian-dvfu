$f_{0}(x) = \frac{1}{2}\left( x^{T} Ax \right) + b \cdot x$
$g(x) = ||x - x_{0}|| - r$
$A = 3 \times 3$. Невырожденная

# Решение

$\sqrt{ (x - x_{0}) } - r \leq 0$
$\sqrt{ (x - x_{0}) } \leq r$
$x - x_{0} - r^{2} \leq 0$
