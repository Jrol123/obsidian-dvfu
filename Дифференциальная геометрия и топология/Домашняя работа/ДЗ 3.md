# Задача
Доказать, что $\Sigma$ является канонической топологией на $\mathbb{R}^{2}$
# №1
## Условие
$\Sigma^{2}$ - все открытые круги
$B^{2} ((x_{0}, y_{0}), \epsilon) = \left\{ (x, y) | (x - x_{0})^{2 + (y - y_{0}^{2 < \epsilon})} \right\}$
## Решение
